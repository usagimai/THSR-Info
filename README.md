## 【作品說明】

### ● 名稱：

台灣高鐵對號座開賣資訊 APP

### ● 簡介：

1. 查詢台灣高鐵對號座訂位開放時程
   - 從乘車日期回推開賣日
   - 查詢特定日期可購買之日期區間
2. 提供台灣高鐵最新資訊

### ● 使用技術/工具：

React.js、MobX、Axios、Material UI

### ● 特色：

1. 使用 MobX 進行 state management (推算日期功能 & 高鐵最新消息資料)
2. 多數操作介面使用 Material UI (Tab、日期選擇、表格、Pagination 等)
3. 使用 Axios 串接公共運輸整合資訊流通服務平台的高鐵最新消息 API
